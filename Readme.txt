# CucumberBDD---Selenium-Java
Hybrid framework with POM, Cucumber BDD and basics of selenium with Java

Below are things which I have understood and covered in this framework
-Feature file mapping with step definition
-regex in cucumber
-Tags
-Cucumber Hooks
-Background scenario
-Scenario outline
-Different reports in cucumber BDD
-Data table as lists
-Datatacle as Maps
-Running in parallel - Junit
-Running scenarios in parallel - TestNG
-Skip a scenario in cucumber
-Serenity integration for reports - run in cmd using maven - "mvn clean verify" and get serenity BDD report
